# Oldi-Berdi

A regional classified marketplace platform featuring multi-level hierarchical location filtering and an adaptive web interface.

### 🚀 Key Features
* **Hybrid Architecture:** Combines efficient server-side rendering (Thymeleaf) with RESTful APIs for dynamic data loading.
* **Database Design (PostgreSQL):** Transformed unstructured category-specific data into a structured relational model using a flexible attributes-options schema (`categories_attributes`, `options`).
* **Advanced Filtering:** Engineered robust entity relationships for multi-level locations (provinces/cities) and implemented customized server-side pagination.
* **Clean Code & Architecture:** Built on solid backend principles utilizing Service-Mapper layers and central global exception handling.
* **Modern UI/UX:** Features an adaptive and attractive interface styled with Glassmorphic elements and Bootstrap.

### 🛠 Tech Stack
* **Backend:** Java, Spring Boot, Spring Web (REST), Spring Data JPA, Hibernate
* **Database:** PostgreSQL
* **Frontend:** Thymeleaf, Bootstrap, HTML5, CSS3, JavaScript
